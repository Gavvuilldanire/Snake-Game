# Snake Game (C++)

**Author:** Gantar
**Environment:** Embarcadero IDE

## Deskripsi
Implementasi permainan klasik Snake berbasis konsol (CLI). Proyek ini ditulis dalam bahasa C++ dan dirancang spesifik untuk dijalankan pada lingkungan pengembangan **Embarcadero**. Fokus utama pengembangan adalah stabilitas *rendering* dan manajemen logika permainan.

## Fitur Utama
* **Player Identity:** Input nama pemain di awal sesi.
* **High Score System:** Melacak dan menyimpan skor tertinggi.
* **Stable Rendering:** Visualisasi pergerakan ular tanpa *flickering* (kedipan layar) yang mengganggu.
* **Logic Control:** Batasan pergerakan logis (tidak bisa berbalik arah 180Â° secara instan).

## Kebutuhan Sistem
* Software: Embarcadero C++ Builder / Dev-C++
* OS: Windows (Menggunakan `windows.h` API)

## Cara Menjalankan
1.  Buka *source code* pada software Embarcadero.
2.  Lakukan *Compile* untuk memastikan tidak ada *error syntax*.
3.  *Run* program melalui terminal/konsol bawaan.
